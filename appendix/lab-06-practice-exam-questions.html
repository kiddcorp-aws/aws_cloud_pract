Domain 1: Cloud Concepts (24%)

Cloud computing fundamentals and AWS value propositions

Cloud Concepts

Question 1

Which of the following is a characteristic of cloud computing as defined by NIST?

  1. Fixed monthly pricing regardless of usage
  2. On-demand self-service
  3. Requires long-term contracts
  4. Single geographic location
Correct Answer: B — The five essential characteristics of cloud computing per NIST are: on-demand self-service, broad network access, resource pooling, rapid elasticity, and measured service.
Cloud Concepts

Question 2

A company wants to reduce the capital expenditure (CapEx) of maintaining on-premises data centers. Which benefit of cloud computing addresses this?

  1. Dedicated Hosts
  2. Trading fixed expense for variable expense
  3. Increased hardware control
  4. Longer procurement cycles
Correct Answer: B — One of the six advantages of cloud computing is trading fixed expense for variable expense. Instead of investing heavily in data centers before knowing usage, you pay only when you consume resources.
Cloud Concepts

Question 3

Which cloud computing model provides the customer with the most control over the underlying infrastructure?

  1. Software as a Service (SaaS)
  2. Platform as a Service (PaaS)
  3. Infrastructure as a Service (IaaS)
  4. Function as a Service (FaaS)
Correct Answer: C — IaaS provides the most control. The customer manages the OS, middleware, runtime, data, and applications. AWS EC2 is an example of IaaS.

Domain 2: Security and Compliance (30%)

Shared responsibility model, IAM, and security services

Security

Question 4

Under the AWS Shared Responsibility Model, which of the following is the customer's responsibility?

  1. Physical security of data centers
  2. Patching the hypervisor
  3. Configuring security group rules
  4. Maintaining network infrastructure hardware
Correct Answer: C — Configuring security groups is the customer's responsibility (security IN the cloud). AWS is responsible for the physical infrastructure, hypervisor, and network hardware (security OF the cloud).
Security

Question 5

Which IAM best practice should be implemented immediately after creating a new AWS account?

  1. Delete the root account
  2. Create access keys for the root user
  3. Enable multi-factor authentication (MFA) on the root user
  4. Share root credentials with the team lead
Correct Answer: C — Enabling MFA on the root user is a critical first step. The root account cannot be deleted (A), you should never create root access keys (B), and credentials should never be shared (D).
Security

Question 6

Which AWS service records API calls made in your AWS account for auditing purposes?

  1. Amazon CloudWatch
  2. AWS CloudTrail
  3. AWS Config
  4. Amazon Inspector
Correct Answer: B — AWS CloudTrail records API calls made in your account, providing an audit trail. CloudWatch monitors performance metrics. AWS Config tracks resource configuration changes. Amazon Inspector assesses application security.

Domain 3: Cloud Technology and Services (34%)

Core AWS services and architecture

Technology

Question 7

Which AWS service provides resizable compute capacity in the cloud?

  1. Amazon S3
  2. Amazon EC2
  3. Amazon RDS
  4. Amazon VPC
Correct Answer: B — Amazon EC2 (Elastic Compute Cloud) provides resizable compute capacity. S3 is object storage, RDS is managed databases, and VPC is networking.
Technology

Question 8

A company needs to store infrequently accessed data at a lower cost but requires millisecond access when retrieved. Which S3 storage class should they use?

  1. S3 Standard
  2. S3 Standard-Infrequent Access (S3 Standard-IA)
  3. S3 Glacier Deep Archive
  4. S3 One Zone-Infrequent Access
Correct Answer: B — S3 Standard-IA is designed for data that is accessed less frequently but still requires rapid access (milliseconds). Glacier Deep Archive has retrieval times of 12+ hours. One Zone-IA stores data in only one AZ, providing less durability.
Technology

Question 9

Which component of Amazon VPC acts as a stateful firewall at the instance level?

  1. Network Access Control List (NACL)
  2. Route Table
  3. Security Group
  4. Internet Gateway
Correct Answer: C — Security groups are stateful firewalls at the instance (ENI) level. NACLs are stateless and operate at the subnet level. Route tables direct traffic, and internet gateways provide internet access.
Technology

Question 10

Which AWS service is a fully managed NoSQL database that provides single-digit millisecond performance at any scale?

  1. Amazon RDS
  2. Amazon Redshift
  3. Amazon DynamoDB
  4. Amazon Aurora
Correct Answer: C — Amazon DynamoDB is a fully managed NoSQL database offering single-digit millisecond latency. RDS and Aurora are relational databases. Redshift is a data warehouse.
Technology

Question 11

Which AWS service allows you to define your infrastructure as code using JSON or YAML templates?

  1. AWS Elastic Beanstalk
  2. AWS CloudFormation
  3. AWS CodeDeploy
  4. Amazon CloudWatch
Correct Answer: B — AWS CloudFormation enables Infrastructure as Code using JSON or YAML templates. Elastic Beanstalk is a PaaS that deploys applications. CodeDeploy automates application deployments. CloudWatch is for monitoring.

Domain 4: Billing, Pricing, and Support (12%)

Pricing models, billing tools, and support plans

Billing

Question 12

Which EC2 pricing model offers the largest discount (up to 90%) compared to On-Demand pricing but can be interrupted by AWS with a 2-minute warning?

  1. Reserved Instances
  2. Savings Plans
  3. Spot Instances
  4. Dedicated Hosts
Correct Answer: C — Spot Instances can save up to 90% but can be reclaimed by AWS with a 2-minute notification. They are ideal for fault-tolerant, flexible workloads like batch processing.
Billing

Question 13

Which AWS tool allows you to set custom budgets and receive alerts when costs or usage exceed your thresholds?

  1. AWS Cost Explorer
  2. AWS Budgets
  3. AWS Pricing Calculator
  4. AWS Trusted Advisor
Correct Answer: B — AWS Budgets lets you set custom cost and usage budgets with email or SNS alerts. Cost Explorer visualizes historical costs. Pricing Calculator estimates future costs. Trusted Advisor provides optimization recommendations.
Billing

Question 14

Which AWS Support plan provides access to a Technical Account Manager (TAM)?

  1. Basic
  2. Developer
  3. Business
  4. Enterprise
Correct Answer: D — Only the Enterprise Support plan (and Enterprise On-Ramp) includes a designated Technical Account Manager (TAM). Basic and Developer plans do not include TAM access. Business provides 24/7 support but no TAM.
Billing

Question 15

Data transfer INTO AWS from the internet is:

  1. Charged at a premium rate
  2. Free of charge
  3. Charged per GB after the first 100 GB
  4. Only free for S3 transfers
Correct Answer: B — Data transfer into AWS from the internet is free across all services. Data transfer out from AWS to the internet is where charges apply.
