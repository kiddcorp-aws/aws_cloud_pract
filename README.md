# AWS Cloud Practitioner — 1-Day Course

## Course Overview
This intensive 1-day course prepares participants for the AWS Certified Cloud Practitioner exam (CLF-C02). It covers the four domains tested on the exam: Cloud Concepts, Security and Compliance, Cloud Technology and Services, and Billing and Pricing.
